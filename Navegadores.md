# GENERALES
- Abrir nueva pestaña: Ctrl + T (Windows/Linux) / Cmd + T (Mac).
- Cerrar pestaña actual: Ctrl + W / Cmd + W.
- Reabrir pestaña cerrada: Ctrl + Shift + T / Cmd + Shift + T.
- Ir a la barra de direcciones: Ctrl + L / Cmd + L.
- Buscar en la página: Ctrl + F / Cmd + F.
- Historial: Ctrl + H / Cmd + Y.
- Descargas: Ctrl + J / Cmd + Shift + J.
- Modo Incógnito/Privado: Ctrl + Shift + N / Cmd + Shift + N.
- Recargar página: F5 o Ctrl + R.
